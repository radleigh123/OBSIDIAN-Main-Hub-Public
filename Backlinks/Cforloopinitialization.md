## For loop
> It’s okay to not include or declare again both the initialization and condition. Simply declare only the ‘`;`’ to let the compiler know.

```C
int i = 1024;
for(; i; i >>= 1) {
	printf("Hello World\n"); // prints 11 times
}
```

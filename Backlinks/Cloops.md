# Loop
- **do... while()** loop
```C
do {
	statement1;
} while(expression);
```

- **while()** loop
```C
while(expression) {
	statement1;
	statement2;
}
```

- **for()** loop
```C
for(initialization; condition; increment) {
	statement1;
}
```

### Loop control statements:
> **Break** - used to terminate from the loop.

> **Continue** - it forces to execute the next iteration of the loop.

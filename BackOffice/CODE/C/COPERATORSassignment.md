## Assignment Operator
>[!recite|no-i collapse clean] (=, +=, -=, \*=, /=, %=) (<<=, >>=, &=, |=, ^=)

>[!column|no-t flex]
>>[!NOTE|clean no-t]
>>- is [[COPERATORSbinary|binary operator]]
>>- requires 2 values (lvalue & rvalue)
>
>>[!NOTE|clean no-t]
>>- copies rvalue to lvalue
>> e.g. `var = 5;`
>>- which used to assign values to a variable

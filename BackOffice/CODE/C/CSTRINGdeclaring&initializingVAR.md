## Declaring and Initializing variables

> **String variable**
> is a one dimensional [array](Carrays.md) of characters that is capable of holding a string at a time.
> `char string1[6]`

```C
char string[6] = "Hello"; //not a string literal
//char string[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
```

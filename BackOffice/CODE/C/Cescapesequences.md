# Escape sequences
|        | Meaning            |
| ------ | ------------------ |
| `\a`   | Alarm/Beep         |
| `\b`   | Backspace          |
| `\f`   | Form Feed          |
| `\n`   | New Line           |
| `\r`   | Carriage Return    |
| `\t`   | Tab (Horizontal)   |
| `\v`   | Vertical Tab       |
| `\\`   | Backslash          |
| `\`    | Single Quote       |
| `\"`   | Double Quote       |
| `\?`   | Question Mark      |
| `\nnn` | Octal Number       |
| `\xhh` | Hexadecimal Number |
| `\0`   | Null               |

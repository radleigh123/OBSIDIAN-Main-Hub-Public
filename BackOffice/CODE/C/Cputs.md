## `puts()`
> is a function declared in **<stdio.h> library** and is used to write strings to the output screen.

> Also, puts() function automatically writes a newline character after writing the string to the output screen. 

```C
char *s = "Hello";
puts(s);
puts(s);

/*
Hello
Hello
*/
```

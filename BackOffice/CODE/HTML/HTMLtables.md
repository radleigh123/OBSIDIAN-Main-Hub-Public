# Table element `<table>`
**e.g.**
```HTML
<table>  
  <tr>  
    <th>Company</th>  
    <th>Contact</th>  
    <th>Country</th>  
  </tr>  
  <tr>  
    <td>Alfreds Futterkiste</td>  
    <td>Maria Anders</td>  
    <td>Germany</td>  
  </tr>   
</table>
```
![[Pasted image 20221124195718.png|center]]

**`<td></td>` Table Cells** - stands for table data.
**`<tr></tr>` Table Rows** - stands for table rows
## Table Headers `<th> </th>`
```HTML
<table>  
  <tr>  
    <th>Person 1</th>  
    <th>Person 2</th>  
    <th>Person 3</th>  
  </tr>  
  <tr>  
    <td>Emil</td>  
    <td>Tobias</td>  
    <td>Linus</td>  
  </tr>  
  <tr>  
    <td>16</td>  
    <td>14</td>  
    <td>10</td>  
  </tr>  
</table>
```

## `<q>` inline quotation element
The HTML **`<q>`** tag defines a short quotation.
> Browsers normally insert quotation marks around the quotation.

```HTML
<p>
WWF's goal is to: 
<q>Build a future where people live in harmony with nature.</q>
</p>
```

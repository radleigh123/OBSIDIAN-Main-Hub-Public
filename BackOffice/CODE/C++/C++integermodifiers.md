## Integer Modifiers
```cpp
#include <iostream>

int main()
{
    short short_var{-32768};

    unsigned int unsigned_var{456};

    long long_var{88};

    long long long_long_var{888};

    return 0;
}
```

>[!WARNING|alt-co] These modifiers only apply to integral types: those in which can store decimal values

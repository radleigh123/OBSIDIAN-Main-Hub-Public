>[!CHECK]- # <u>CSS</u> (Cascade Stylesheeting)

>- describes how HTML elements should be displayed.

> 	background-color: lightblue;
> }
>   
> h1 {
> 	color: white;  
> 	text-align: center;
> }
> 
> p {
> 	font-family: verdana;  
> 	font-size: 20px;
> }
> ```

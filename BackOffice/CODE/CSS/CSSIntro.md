>[!CHECK] # <u>CSS</u> (Cascade Stylesheeting)
>- is the language we use to style an HTML document.
>- describes how HTML elements should be displayed.
>```CSS
> body{
> 	background-color: lightblue;
> }
>   
> h1 {
> 	color: white;  
> 	text-align: center;
> }
> 
> p {
> 	font-family: verdana;  
> 	font-size: 20px;
> }
> ```
